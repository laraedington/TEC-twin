Readme for TEC-twin MATLAB code
=================================

Includes following files:

********************************** Files **************************************************************

README.txt              - This file.

Duffing_Case1_TEC.m     - The script for applying TEC-twin framework to Duffing oscillator in Case 1.

Duffing_Case2_TEC.m     - The script for applying TEC-twin framework to Duffing oscillator in Case 2.

Tanks_TEC.m             - The script for applying TEC-twin framework to Cascading Tanks system.
			  Requires data available at: https://doi.org/10.4121/12960104.v1

duffing.m               - Functions describing Duffing oscillator to be integrated by ode45 solver.

nonlinear.m             - Functions describing nonlinear oscillator to be integrated by ode45 solver.

tanks.m          	- Functions describing Cascading Tanks system to be integrated by ode45 solver.

********************************************************************************************************
